# Changelog

## v0.1.0 - Initial public release

Initial public baseline for dukememory:

- local-first project memory over PostgreSQL and `pgvector`;
- hybrid keyword + semantic retrieval;
- Ollama-backed embeddings, extraction, validation, and optional reranking;
- `dukememory_*` MCP server tools for memory, context, graph, code, audit,
  backup, import/export, eval, and maintenance workflows;
- project-scoped code symbol index and approximate call graph navigation;
- pending/active/superseded/archived memory lifecycle;
- Codex config and hook generation;
- native memory/code graph viewer;
- production, MCP, Codex, hook, and retrieval eval checks.
